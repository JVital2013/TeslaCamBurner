# TeslaCamBurner

Encode Tesla Dashcam Clips with telemetry data in the video itself, for easy sharing and distribution. To use, pull clips from the glovebox USB drive from a car running the 2025 Holiday Update or newer.



Windows Builds are available [here](https://github.com/JVital2013/TeslaCamBurner/releases). Extract the zip file, and run "Tesla Cam Burner GUI.exe". Requires .NET 8.0
